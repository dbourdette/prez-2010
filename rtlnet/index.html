RTL Net

RTL net

RTL net est la filliale internet de RTL France (RTL, RTL 2, FUN RADIO).

RTL net développe les sites des 3 radios :

  • www.rtl.fr
  • www.rtl2.fr
  • www.funradio.fr

www.rtl.fr est le premier site de radio généraliste de France et héberge les blogs de nombreuses personnalités de l'audiovisuel.

RTL net est aussi présent sur les smartphones mais pas en tant qu'éditeur de logiciels.

Equipes

Les demandes sont faites par

  • Les responsables des sites thématiques
  • Les éditeurs
  • Le marketing
  • Les commerciaux
  • Les services premium

La réalisation est assurée par

  • Un chef de projet
  • Le studio (graphistes et webmasters) (5 personnes)
  • L'équipe de développement (5 personnes)
Process

Le développement est conduit par un processus agile (agile manifesto) dérivé de la méthodologie scrum.

Ce processus est itératif (2 semaines) et suit les valeurs de l'agilité

  • Intéractions et collaboration entre les individus
  • La valeur est dans le logiciel qui fonctionne
  • Introspection et amélioration continue (kaizen)

Le processus évolue continuellement.

Outils

Les outils de l'usine logicielle sont les suivants

  • jira : outils de ticketing
  • eclipse : environnement de développement
  • maven : outils de build et descripteur de projet
  • subversion : outils de gestion de sources
  • jenkins : intégration continue
  • nexus : dépot de binaires
  • rundeck : automisation des déploiements
Technologies

Les serveurs utilisés en production

  • solaris : système d'explotation des serveurs de prod
  • tomcat : serveurs d'application
  • varnish : proxy et cache htttp
  • oracle : base de données relationnelles
  • solr : moteur de recherche

Les principales librairies utilisées dans les développements

  • spring framework
  • hibernate
  • ehcache et jboss cache
  • apache commons
  • junit
